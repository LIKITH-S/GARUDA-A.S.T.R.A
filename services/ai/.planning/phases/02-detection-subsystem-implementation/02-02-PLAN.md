  <action>
    Create and implement `services/ai/detection/face_cropper.py` for safe boundary slicing:
    - Define a function `crop_face(frame: np.ndarray, facial_area: list) -> np.ndarray` that accepts the image array and bounding box coordinates `[x1, y1, x2, y2]`.
    - Extract image height and width using `frame.shape[:2]`.
    - Clamp coordinates:
      - `x1 = max(0, int(facial_area[0]))`
      - `y1 = max(0, int(facial_area[1]))`
      - `x2 = min(width, int(facial_area[2]))`
      - `y2 = min(height, int(facial_area[3]))`
    - Perform array slicing `frame[y1:y2, x1:x2]` and return the cropped face image array (preserving BGR color format).
    - Wrap in try-except to log errors and return None or empty array if coordinates are invalid or array slicing fails.
  </action>
