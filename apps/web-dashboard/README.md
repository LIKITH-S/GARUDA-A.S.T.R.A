# Garuda Astra | Advanced Surveillance & Response Platform

This is a high-premium frontend replica of the Garuda Astra Command Center, built with Next.js 15+ and Tailwind CSS v4.

## Features
- **Real-time Dashboard:** Hero match cards, live activity logs, and mission metrics.
- **Live Alerts:** Critical incident monitoring with confidence level detection.
- **Surveillance Uploads:** Batch ingestion console for footage and images.
- **Detection Logs:** Searchable audit trail of all AI-processed events.
- **Missing Persons Registry:** Centralized database for active search operations.
- **Patrol Unit Management:** Tactical tracking of unit health and deployment.
- **Response Board:** Kanban-style coordinator for mission response.

## Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Design:** Tactical Dark Mode with Glassmorphism

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
